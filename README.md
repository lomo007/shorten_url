# 縮網址器

## 介紹

縮短長網址

### 功能

- 可輸入要轉換的網址
- 產生一組短網址

## 開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入:

   ```bash
   cd restaurant_list
   ```

4. 若看見'mongodb connected'表示伺服器連線成功。若看見'done'表示種子資料載入成功。
   輸入'ctrl + c'停止:
   
   ```bash
   mongodb connected!
   mongodb connected!
   done
   ```

5. 接著輸入'npm run start'執行渲染畫面:

   ```bash
   npm run start
   ```

6. 若看見"Express is listening on http://localhost:3000"
   則代表順利運行打開瀏覽器進入到以下網址:

   ```bash
   http://localhost:3000
   ```

7. 若欲暫停使用

   ```bash
   ctrl + c
   ```
